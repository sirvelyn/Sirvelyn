# Sirvelyn v0.1.0 — Rilis Perdana 🎉

Terminal emulator bergaya **pixel-art** untuk Windows. Ringan, cepat, dan enak dipakai sehari-hari — dibangun dengan Tauri 2 + React 19 + xterm.js.

## ✨ Fitur Utama

**Terminal**
- Multi-terminal dengan tab — buka beberapa shell sekaligus
- Pemilih shell otomatis (PowerShell, CMD, dan shell lain yang terpasang)
- Rendering WebGL — mulus, tanpa "patah-patah" di aplikasi TUI
- Restart terminal yang sudah mati tanpa kehilangan tab-nya

**Layout & Tampilan**
- Split-pane: bagi layar jadi beberapa terminal, dengan pembatas yang bisa digeser
- Ukuran font bisa diatur (Ctrl `+` / `-` / `0`) dan diingat
- Ganti nama tab & workspace langsung di tempat (klik untuk rename)
- Ukuran & posisi window diingat antar sesi
- Custom titlebar yang bersih

**Produktivitas**
- Copy/paste cepat: `Ctrl+Shift+C` / `Ctrl+Shift+V`
- Pintasan keyboard: tab baru, tutup, pindah/lompat antar terminal
- Panel ⚙ Pengaturan lengkap dengan panduan pintasan

**Pembaruan**
- Auto-update terintegrasi — cek pembaruan langsung dari dalam aplikasi (⚙ → "Cek pembaruan"). Update ditandatangani secara kriptografis untuk keamanan.

## 📥 Cara Instal

Unduh salah satu dari aset di bawah:
- **`Sirvelyn_0.1.0_x64_en-US.msi`** — installer MSI (disarankan)
- **`Sirvelyn_0.1.0_x64-setup.exe`** — installer NSIS

> Catatan: Karena belum ditandatangani dengan sertifikat berbayar, Windows SmartScreen mungkin menampilkan peringatan saat pertama kali instal. Klik **"More info" → "Run anyway"** untuk melanjutkan.

## 💻 Persyaratan

- Windows 10 / 11 (64-bit)
